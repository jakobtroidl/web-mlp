# WONNX WASM using Vite

An example of how to use [WONNX](https://github.com/webonnx/wonnx) on the web though WebAssembly. This uses [vite](https://vitejs.dev) as the bundler.

## Running

```sh
npm install
npm run dev
```

## Building

```sh
npm run build
```
