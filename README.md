# WebGPU accelerated fast MLP inference

## Install
```
npm i web-mlp
```

## Running

```sh
npm install
npm run dev
```

## Building

```sh
npm run build
```

## Pubish to npm
```
npm run build
npm pack
npm login
npm publish
```
